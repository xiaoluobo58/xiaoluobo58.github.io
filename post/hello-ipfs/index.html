Hello IPFS

2018-12-12 1 min read # IPFS

👏 欢迎使用 星际文件系统 InterPlanetary File System（IPFS） ！
✍️ 星际文件系统 是一个旨在实现文件的分布式存储、共享和持久化的网络传输协议...

特性👇

📝 你可以使用最酷的 Python 语言，进行快速创作

🌉 你可以在任意位置分享文件

🏷️ 你可以对给文件添加自己的ENS域名

📋 你可以自定义CID，甚至可以创建外部链接

💻 你可以在 Windows，MacOS 或 Linux 设备上使用此协议

🌎 你可以添加HTML网页向世界展示，未来将支持更多方式

💬 你可以进行简单的配置，接入更多静态系统

🇬🇧 你可以使用中文简体或英语

🌁 你可以任意使用静态网站程序，自定义你的站点

🖥 你可以上传任意文件，无需任何费用

🌱 当然 IPFS 还很年轻，有很多不足，但请相信，它会不停向前 🏃

未来，它一定会成为你离不开的伙伴

尽情发挥你的才华吧！

😘 Enjoy~
